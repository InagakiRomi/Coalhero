## 🎮 專案簡介
本遊戲為一款以像素風格呈現的 2D 動作冒險遊戲，是大學時的畢業專題，並成功上架至 Steam。玩家需在敵人與機關分布的場景中完成各種挑戰。

## 👨‍💻 我的職責與技術重點
我擔任本專案的主要程式開發者，負責核心系統與各項功能的設計與實作：

- 遊戲主邏輯、關卡系統、事件觸發、玩家控制、戰鬥機制等核心功能開發
- 串接 Steam SDK，實作成就系統
- 撰寫所有主要的程式碼，並負責Bug 修正
- 使用 Unity 引擎與 C# 開發，熟悉物件導向設計
